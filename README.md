# Esp8266 NeoPixel clock
This project uses the Adafruit NeoPixel library to control a 60 LED NeoPixel ring as a clock. 

Hardware required

* 4 x Neo Pixel 1/4 Rings (15 LED's each)
* 1 x ESP8266 module. I am using the WeMos D1 mini
* some dupont cables.
